# Handwritten-digit-classification
Analysis and Classification of Handwritten Digits Using the Digits Dataset with Machine Learning. This project demonstrate simple machine learning to classify handwritten digits 0 to 9 using digits dataset from scikit-learn.

## Dataset
- Source: sklearn.datasets.load_digits
- Image size: 8x8 pixels
- Total samples: 1797

## Method
- Preprocessing: StandardScaler
- Model: Logistic Regression
- Train-test split: 80% training, 20% testing

## Result
Model accuracy: 97.2%

## Tools
- Python
- Numpy
- Pandas
- Scikit-learns
